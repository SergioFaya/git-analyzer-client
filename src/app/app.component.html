Welcome

With this app you can evaluate the contributions under your different repos hosted on github in a quick and easy way. Some of the features are:

  • To see number of additions and deletions over the week
  • To see the git network chart
  • Display recent activity of issues, pull requests
Instruccions:
1.- Give the github app access to your repositories (you can skip this step, but is necessary to use some features like the recent activity tabs of issues and pull requests)
2.- Login into the app (in order to evaluate the repos we need access to your account)
3.- The login process will be performed using your github credentials (we dont have access to your password only to an access token)
4.- Know you can start using the app,
